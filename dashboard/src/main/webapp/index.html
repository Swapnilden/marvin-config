The MARVIN Release Bot executes the DBpedia Information Extraction Framework (DIEF) in a regular interval to extract the following release groups:

  • The mappings, generic, and wikidata release groups extraction starts on every 10th of each month.
  • The text release group is only published between 2-3 times a year due to its enormous size.
This dashboard gives details about the current release progress, including several steps.

The configuration of MARVIN is public available at gitlab. Further information are described in The New DBpedia Release Cycle: IncreasingAgility and Efficiency in Knowledge ExtractionWorkflows.

DISCLAIMER: The release depends on several external dependencies. Therefore we can not guarantee a full release every month.

Release Steps. Expand to see documentation for each step.

  1. Download Mappings: Download latest DBpedia mappings form mappings.dbpedia.org
  2. Download Ontology: Download latest DBpedia ontology
  3. Download Wikimedia Dumps: Download latest Wikimedia dumps from dumps.wikimedia.org
  4. Extract: Start DBpedia information extraction
  5. Post-Process: Post-processing of extracted data
  6. Backup Unredirected: Backup unredirected original files
  7. Marvin Release: Deploy raw MARVIN release
  8. Parse Marvin Release: Parse raw Marvin release and backup parse-logs
  9. DBpedia Release: Deploy cleaned DBpedia release
